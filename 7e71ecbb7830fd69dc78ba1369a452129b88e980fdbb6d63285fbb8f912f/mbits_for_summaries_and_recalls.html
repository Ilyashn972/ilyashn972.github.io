Plot meaningful bits for selected recalls and a few attempts for gpt4 summaries

The Summaries are generated by gpt4 while asking it to make them longer or shorter.

Extractive recalls are selected by using prompt5 from prevous paper. We use this prompt to find out segments that the subject recalled. For these segments with calculate mean(p-rec). We select top 5 with the highest values.

Abstractive recalls are selected based "caught" segments. We calculate sum of p-rec for caught segments and divide it by C (number of the clauses in the recall). We select top 5 with the highest values.

See code at mbits_for_summaries_and_recalls.py Also look at experiments/build_summaries_of_different_len.py, build_upper_tree_from_recalls.py, top_5_extractive_recalls.py
