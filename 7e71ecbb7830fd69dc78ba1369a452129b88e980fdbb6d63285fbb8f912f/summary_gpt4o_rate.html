Combare mbits rate for summaries of different length

The plots below were generated based on summaries of varying lengths, created using GPT-4o. I instructed GPT-4o to produce a summary with exactly n key points, where n depends on the length of the story.

Key Points Selection

The number of key points is determined as follows:


    max_keypoints = 11
    if max_segment_id > 30:
    max_keypoints = 15
    if max_segment_id > 50:
    max_keypoints = 25
    if max_segment_id > 100:
    max_keypoints = 30
    if max_segment_id > 150:
    max_keypoints = 35

For each specified number of key points, I generated three different summaries using temperature = 1.

Prompt Used


    summary_with_specified_number_of_key_points = """
    Summarize the provided narrative into exactly {{n}} very simple key points, each containing a subject and a predicate, told from the first-person perspective.
    Format the output in JSON as follows:
    
    {
      "keypoints": ["keypoint1", ..., "keypoint{{n}}"]
    }
    """
