Comparing MBits Prompts

Usually, we use the prompt "The following two texts separated by '---' tell the same story with different wording:" to separate between a recall and the narrative. In this experiment, we wanted to check what will happen if only "\n---\n" separates the recall and the narrative. The plots below also include comparisons:

  1. Narrative vs (Recall + "---" + Narrative)
  2. Rephrased Narrative vs (Recall + "---" + Rephrased Narrative)
  3. Narrative vs (Summary + "---" + Narrative)
  4. Rephrased Narrative vs (Summary + "---" + Rephrased Narrative)
