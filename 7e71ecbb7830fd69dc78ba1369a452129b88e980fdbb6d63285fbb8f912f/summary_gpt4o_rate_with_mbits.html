Interactive Plot with Summary Mbits

The plots below were generated based on summaries of varying lengths, created using GPT-4o. I instructed GPT-4o to produce a summary with exactly n key points, where n depends on the length of the story.

Prompts Used


            summary_with_specified_number_of_key_points = """
            Summarize the provided narrative into exactly {{n}} very simple key points, each containing a subject and a predicate, told from the first-person perspective.
            Format the output in JSON as follows:
            
            {
              "keypoints": ["keypoint1", ..., "keypoint{{n}}"]
            }
            """
            rephrase_narrative = """
            You will be given a narrative, segmented into numbered linguistic clauses, formatted as a JSON object where each key
            corresponds to a clause number, and the value is the clause text.
            
            Your task is to generate a paraphrase of the narrative while strictly maintaining the original meaning. To achieve this,
            you must:
            • Use different wording (lexical diversity) and different sentence structures (syntactic diversity).
            • Ensure strict clause-by-clause paraphrasing, meaning each clause should be paraphrased independently without merging
            or splitting clauses.
            • Strive to keep each clause as close as possible to its original length, allowing only minimal variation.
            • Preserve the numbering of the clauses in the paraphrased output.
            
            Expected Output Format (JSON example):
            
            { "1": "The sun dipped below the horizon.", "2": "A gentle wind moved through the trees." }
