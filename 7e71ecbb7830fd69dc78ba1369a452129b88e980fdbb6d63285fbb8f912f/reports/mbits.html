Mbits report

Meaningful bits.

Below are results related to meaningful bits. If not stated otherwise meaningful bits is difference between sum of logp for segments. The difference is calculated by substracting sum of logp probalibities of the narrative without preceding text from sum of logp probabilities with narrative preceeded by recall.
Prompts:

            {{ preceding_text1 }}

  • Meaningful Bits of Recall

    CSV files containing data for each segment of the narrative.
    These files include:

    • A column for each segment. For example for boyscout from 1 to 19.
    • The total sum of meaningful bits for the full recall.
    • The meaningful bits rate (total sum divided by character length).
    • A row with average values across all segments.

  • Scatter Plot of Meaningful Bits

    This plot shows:

    • The recall length on the x-axis.
    • The meaningful bits rate on the y-axis.
    • A comparison with a one summarized and one rephrased version of the narrative.

  • CSVs of P-caught, P-rec, and Meaningful Bits

    These files contain narratives split into segments. For each segment, the following values are provided:

    • P-caught
    • P-rec
    • Meaningful bits (mbits)
    • The segment text itself
