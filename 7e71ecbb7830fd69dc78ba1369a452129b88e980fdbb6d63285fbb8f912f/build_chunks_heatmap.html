How these plots are created

The heatmaps are generated offline by experiments/build_chunks_heatmap.py using matplotlib, saved as PNGs under html/<story>/chunks_with_recalled_segments_and_fun_out.png, and then embedded on this page with standard <img> tags. There is no in-browser plotting here.

What the heatmap shows

  • Rows: recall_id (ordered roughly by how many segments were recalled).
  • Columns: story segment_id (1..max).
  • Cell value: summed fan_out for that recall and segment, transformed to log1p. Zeros are stored as NaN so they render as missing.

Plot styling/annotations

  • Heatmap rendering: plt.imshow(..., cmap="cool", aspect="auto", vmin=log1p(1)) plus a colorbar labeled ln(fan_out).
  • Recalled segments: black star markers are overlaid at recalled segment positions.
  • Chunk separators: x-axis tick labels come from “level 1” keypoints in key_points.csv.
  • Hover tooltips (local only): when run interactively, mplcursors adds hover annotations; this is not part of the static HTML page.
